# Лабораторная работа 3

Тема: синхронизация потоков с помощью критических секций и событий.

```powershell
cmake -S . -B build-lab3
cmake --build build-lab3 --config Release
build-lab3\bin\Release\Lab3.exe
```

Потоки `marker` заполняют общий массив, сообщают о невозможности продолжения и ожидают от `main` команду продолжить работу или завершиться.
